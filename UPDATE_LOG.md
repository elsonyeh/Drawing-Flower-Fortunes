# 更新日誌 - 遊戲化抽卡系統

## 🎮 v2.0 - 完整抽卡遊戲系統 (2026-01-20)

### ✨ 新增功能

#### 1. 稀有度系統
- 新增 **5 張 SSR 稀有花卡**：
  - 曇花 - 瞬間的永恆（金色漸層）
  - 藍色妖姬 - 奇蹟與不可能（藍金色漸層）
  - 鳳凰花 - 離別與重逢（火紅金漸層）
  - 彼岸花 - 輪迴與重生（紅黑金漸層）
  - 虞美人 - 絢爛與哀愁（紫金漸層）
- **抽卡機率**：每張 SSR 5% 機率（總 25%），一般卡 75%
- SSR 卡片擁有特殊金色漸層配色和動畫效果

#### 2. 遊戲化抽卡動畫
- **多卡洗牌效果**：7 張卡片洗牌展開動畫
- **抽卡過程**：
  - Stage 1: 卡片洗牌（1.5秒）
  - Stage 2: 抽出一張卡（1秒）
  - Stage 3: 卡片翻轉（1.2秒）
  - Stage 4: 花朵綻放（2-2.5秒）
- **SSR 特殊效果**：
  - 金色光芒射線
  - 爆炸式粒子效果
  - 特殊音效提示（未來可添加）

#### 3. 獨特卡背設計
- **每種花擁有獨特卡背**：
  - 基於花朵顏色的漸層背景
  - 獨特的圖案和裝飾
  - 動態花朵符號動畫
- **SSR 卡背特效**：
  - 金色流光動畫
  - 閃爍星星粒子
  - 雙層金邊裝飾
  - SSR 徽章標識

#### 4. 收藏系統
- **完整圖鑑頁面**：
  - 右上角圖鑑按鈕
  - 收集進度統計（總收集率、SSR收集、一般收集）
  - 分類標籤（全部/SSR/一般）
  - 已收集卡片展示卡背
  - 未收集卡片顯示剪影+鎖
  - NEW 標籤標示最新收集
- **localStorage 存儲**：
  - 自動保存抽到的卡片
  - 收集時間記錄
  - 持久化收藏數據
- **圖鑑互動**：
  - 點擊已收集卡片查看詳情
  - 卡片 hover 效果
  - 響應式網格布局

#### 5. 增強首頁動畫
- **60 片飄落花瓣**：
  - 真實花瓣形狀（CSS clip-path）
  - 隨機大小（8-20px）
  - 隨機速度（6-14秒）
  - 旋轉動畫（-360° 至 360°）
  - 擺動軌跡模擬真實飄落
- 保留原有粒子背景效果
- 圖鑑按鈕動畫入場

#### 6. 優化結果頁面
- **3D 花朵為主體**：花朵展示區域加大
- **SSR 專屬效果**：
  - 背景旋轉光線
  - 浮動粒子
  - 金色光暈
  - SSR 徽章
  - 恭喜獲得 SSR 動畫提示
- **分級配色方案**：
  - SSR：金橙色漸層主題
  - 一般：原有花朵顏色主題
- 所有區塊支持稀有度配色

## 🎯 使用說明

### 抽卡流程
1. 首頁點擊「抽出今夜的花語」
2. 觀看洗牌動畫
3. 卡片抽出並翻轉
4. 花朵綻放展示
5. 自動跳轉結果頁（SSR 有特殊慶祝動畫）
6. 卡片自動保存至圖鑑

### 圖鑑使用
1. 點擊右上角圖鑑按鈕
2. 查看收集進度統計
3. 切換分類標籤篩選
4. 點擊已收集卡片查看詳情
5. 持續抽卡解鎖完整圖鑑

### 收集目標
- 🎯 收集全部 20 張花語卡
- ⭐ 收集全部 5 張 SSR 稀有卡
- 📚 達成 100% 圖鑑完成度

## 📝 開發備註

### 機率系統
```javascript
// SSR: 5% × 5 = 25%
// 一般: 75% / 15 ≈ 5% each
0-5%: 曇花
5-10%: 藍色妖姬
10-15%: 鳳凰花
15-20%: 彼岸花
20-25%: 虞美人
25-100%: 一般卡片（隨機）
```
