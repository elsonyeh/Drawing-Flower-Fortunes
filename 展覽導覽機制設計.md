### 🔒 卡池解鎖邏輯

```
進場 → 免費 1 抽（A池）
         ↓
   [A區掃描 ≥2 件] → 解鎖 B池 + 獲得抽卡次數
         ↓
   [B區掃描 ≥2 件] → 解鎖 C池 + 獲得抽卡次數
         ↓
   [C區任意作品] → 完整體驗
         ↓
   [全15件] → 解鎖隱藏SSR
```

**軟性引導（不強硬卡關）：**
- B池未解鎖時仍可抽，但系統提示：「*這朵花還在沉睡⋯⋯它在等你先感受A區的呼吸*」，並抽出A池的卡作為暗示
- C池同理，不報錯、給提示與A/B補充卡

## 二、導覽機制

### 📍 花卡引導設計

每張花卡背面（結果頁）包含：

```
🌸 向日葵
「炎赫的光，從未熄滅」

📍 這朵花正在等你：
   A3 — 炎赫
   📌 千葉素食館外
   「走向那團光，讓它照亮你」
```

- **引導但不強迫**：推薦地點是「邀請」語氣，不是命令
- **距離感知**：可附加「距離現在大約 X 分鐘步行」
- **故事碎片**：每到一個作品掃描後，解鎖該花的完整花語故事

### 🗺️ 自然依序走 A→B→C

| 手段 | 作法 |
|------|------|
| **卡池引力** | A池卡的推薦地點全在A區，自然往A走 |
| **解鎖感** | 掃完2件A，「✨ 蔓延卡池已開啟」出現通知，製造儀式感 |
| **地圖視覺** | 集卡頁顯示花園地圖，A區亮→B區亮→C區亮，視覺進度 |
| **QR回饋** | 掃A區QR → 獲得抽卡 + 一句「B區正在蔓延，等你走近」 |

### ✅ 確保每區看 ≥2 件

| 機制 | 說明 |
|------|------|
| **掃描記錄** | QR code 帶 `?zone=A&work=A3` 參數，系統記錄哪件被掃過 |
| **進度條** | 集卡頁顯示 `A區：2/5 ✓`、`B區：0/5 🔒` |
| **抽卡次數綁掃描** | 每掃一件 QR = +1 抽卡次數，不掃就沒得抽，自然激勵走動 |
| **不重複獎勵** | 同一件作品掃第二次：給花語故事補充，但不給額外抽次 |

## 三、回饋與誘因

### 掃描 QR Code 後的回饋流程

```
掃描 QR Code
     ↓
📖 作品簡介彈窗（5秒）
「炎赫——熱度從不是問題，是一種語言」
     ↓
🎁 獲得 +1 抽卡次數（動畫提示）
     ↓
🌸 解鎖此作品的「花語片段」
（集滿同區可拼出完整花語詩）
     ↓
（若達到區域門檻）✨ 下一卡池解鎖通知
```

### 誘因設計

| 層次 | 誘因 | 觸發條件 |
|------|------|--------|
| 即時 | +1 抽卡次數 | 每次掃描 |
| 短期 | 解鎖新卡池 | 掃A≥2、B≥2 |
| 中期 | 集齊一區5張花卡 | 拜訪完整區域 |
| 長期 | 隱藏SSR彼岸花 | 掃全15件 |
| 社交 | 分享花語到社群 | 抽到SSR時 |

**花語詩碎片（蒐集動機）：**

每區5件作品各含一句詩行，蒐集滿5張才能讀完整詩：

> *呼吸是一種語言*（A1）
> *一瞬即永恆*（A2）
> *熱度從不熄滅*（A3）
> *情緒找到了棲身之所*（A4）
> *時間流動如金*（A5）

## 四、完整使用者流程

```
STEP 1 ── 進場
 │  看到入口海報 / 工作人員說明
 │  掃入口 QR Code → 開啟花語解籤 App
 │
STEP 2 ── 首次抽卡（免費 1 抽，A池）
 │  抽到：向日葵
 │  顯示：「A3 炎赫 — 千葉素食館外，走去看看它」
 │
STEP 3 ── 前往 A 區第一件作品
 │  找到 A3，欣賞裝置藝術
 │  掃描作品旁 QR Code
 │  → 獲得作品簡介 + +1 抽卡次數 + 花語片段①
 │
STEP 4 ── 第二次抽卡（A池）
 │  抽到：薰衣草 → 引導往 A4
 │  （或任意選擇去哪件A區作品）
 │
STEP 5 ── 前往 A 區第二件作品
 │  掃描 A4 QR Code
 │  → ✨「蔓延卡池已解鎖！B區正在等你」
 │  → +1 抽卡次數
 │
STEP 6 ── 抽 B 池卡
 │  抽到：繡球花 → 引導往 B2 情緒共振實驗室
 │
STEP 7 ── 探索 B 區（至少 2 件）
 │  掃 B2 → +1 抽 → 繼續走
 │  掃 B3 → +1 抽 → ✨「共生卡池解鎖！」
 │
STEP 8 ── 抽 C 池卡 → 前往 C 區
 │  掃 C1、C3... 依花卡引導
 │
STEP 9 ── 結尾
 │  蒐集頁顯示已拜訪作品 / 花卡 / 詩句
 │  若掃滿15件 → 解鎖 SSR 彼岸花 + 完整花語詩
 │  → 可分享「我的鹽埕花語」圖卡
 │
STEP 10 ── 選配：回到最喜歡的作品拍照留念
```

### 卡池解鎖判斷
```javascript
const unlockB = visitedWorks.filter(w => w.startsWith('A')).length >= 2
const unlockC = visitedWorks.filter(w => w.startsWith('B')).length >= 2
const unlockSSR = visitedWorks.length >= 15
```

## 六、設計原則總結

| 原則 | 作法 |
|------|------|
| **引導不強迫** | 花卡推薦地點是邀請語氣，軟性卡池限制 |
| **即時回饋** | 每掃一件立刻給抽次 + 故事，不延遲獎勵 |
| **進度可見** | 花園地圖、詩句拼圖讓人知道自己走了多遠 |
| **稀有感驅動** | SSR 花卡讓人願意「多走一件」 |
| **低門檻進入** | 1 次免費抽即可開始，不需登入 |
| **收藏感完結** | 完整花語詩 + 分享圖卡提供儀式性結尾 |
