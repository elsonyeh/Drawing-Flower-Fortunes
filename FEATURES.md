# 埕花 v2.0 功能總覽

## 🎮 核心功能

### 1. 遊戲化抽卡系統
參考原神、崩壞等遊戲的抽卡機制打造

**抽卡流程（5-6秒）：**
```
首頁 → 多卡洗牌(1.5s) → 抽出發光(1s) → 卡片翻轉(1.2s) → 花朵綻放(2s) → 結果頁
```

**稀有度系統：**
- **SSR（5張）**：各5%機率 = 25%總機率
- **一般（15張）**：75%總機率

**SSR特殊效果：**
- ✨ 金色光芒射線（12條旋轉）
- 💥 粒子爆炸效果（30個）
- 🌟 閃爍星星特效
- 🎉 恭喜獲得提示

### 2. 獨特卡背設計
每種花都有專屬卡背，非千篇一律！

**一般卡卡背：**
- 基於花朵顏色的漸層背景
- 獨特圓點圖案
- 動態中央花朵符號
- 四角花朵裝飾

**SSR卡背：**
- 金色三色漸層背景
- 流光動畫效果
- 閃爍星星粒子（20個）
- 雙層金邊裝飾
- SSR徽章標識
- 中央旋轉✦符號

### 3. 完整收藏系統

**圖鑑頁面：**
- 📊 收集統計（總收集率、SSR收集、一般收集）
- 🏷️ 分類篩選（全部/SSR/一般）
- 🔒 未收集顯示剪影+鎖
- ✅ 已收集展示卡背
- 🆕 NEW標籤（最新3張）
- 📱 響應式網格（2-5列）

**數據持久化：**
- localStorage自動存儲
- 收集時間記錄
- 統計數據計算

### 4. 首頁花瓣動畫
60片真實花瓣從天飄落

**特性：**
- 真實花瓣形狀（CSS clip-path）
- 隨機大小：8-20px
- 隨機速度：6-14秒
- 旋轉動畫：-360°到360°
- 左右擺動軌跡
- 無限循環

### 5. 3D花朵展示
使用 React Three Fiber + Three.js

**效果：**
- 8片花瓣圍繞中心
- 自動旋轉展示
- 花瓣波動動畫
- 動態扭曲材質
- 基於花朵顏色渲染

### 6. SSR結果頁特效

**背景效果：**
- 旋轉金色光線（8條）
- 浮動粒子（15個）
- 金色光暈脈衝

**視覺設計：**
- 金橙色配色主題
- 漸層文字效果
- SSR徽章
- 恭喜獲得動畫提示

## 🎯 機率系統

```javascript
// 抽卡機率分配
0-5%     → 曇花 (SSR)
5-10%    → 藍色妖姬 (SSR)
10-15%   → 鳳凰花 (SSR)
15-20%   → 彼岸花 (SSR)
20-25%   → 虞美人 (SSR)
25-100%  → 一般卡片（隨機15選1）
```

**SSR期望值：**
- 平均抽4次出1張SSR
- 平均抽20次收集1張特定SSR
- 平均抽100次集齊5張SSR

## 📱 響應式設計

**斷點：**
- 手機：< 640px（2列卡片）
- 平板：640-768px（3列卡片）
- 桌面：768-1024px（4列卡片）
- 大屏：> 1024px（5列卡片）

**優化：**
- 圖鑑按鈕小螢幕只顯示圖示
- 卡片大小自適應
- 觸控友善按鈕尺寸
- 動畫效能優化

## 💡 使用技巧

**收集建議：**
1. 每次抽卡都會自動保存
2. 可以重複抽到已收集的卡片
3. 圖鑑中NEW標籤顯示最新3張
4. 點擊已收集卡片可重新查看

**SSR獲得提示：**
- 抽卡時出現金色光芒
- 卡片翻轉後有金色粒子爆發
- 結果頁顯示SSR徽章
- 恭喜獲得動畫提示

**清空收藏：**
```javascript
// 在瀏覽器控制台執行
localStorage.removeItem('collectedFlowers')
location.reload()
```
