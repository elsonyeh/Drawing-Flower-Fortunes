# 埕花 (Cheng Flowers)

> 讓花語成為今日的指引，帶你走進屬於自己的鹽埕夜晚

鹽夏不夜埕花語抽籤互動網站 - 一個結合數位互動與在地文化的**遊戲化抽卡體驗**。

## ✨ 專案特色

### 🎮 完整抽卡遊戲系統
- 🎴 **多卡洗牌動畫**：從 7 張卡中抽出專屬花語
- ⭐ **SSR 稀有卡片**：5 張特殊稀有花卡，各 1% 機率
- 🎨 **獨特卡背設計**：每種花都有專屬的卡背圖案和配色
- ✨ **SSR 特殊效果**：金色光芒、粒子爆發、專屬動畫
- 📚 **完整收藏系統**：圖鑑追蹤、收集進度、localStorage 存儲

### 🌸 豐富內容
- **20 張花語卡片**：15 張一般卡 + 5 張 SSR 稀有卡
- **在地化內容**：結合鹽埕實際地點的探索建議
- **花語故事**：每朵花都有獨特的意義、故事與訊息

### 🎬 極致動畫體驗
- **60 片飄落花瓣**：首頁真實花瓣飄落動畫
- **2D + 3D 混合**：卡片翻轉 + 立體花朵綻放
- **流暢轉場**：5-6 秒完整抽卡過程動畫
- **響應式設計**：手機/平板/桌面完美適配

### ⚡ 技術亮點
- **React 18 + Vite**：快速開發和建置
- **Framer Motion**：流暢的 2D 動畫
- **React Three Fiber**：3D 花朵渲染
- **localStorage**：持久化收藏數據

## 🎯 使用流程

### 抽卡體驗
1. **進入首頁** - 欣賞 60 片花瓣飄落動畫
2. **點擊抽卡** - 「抽出今夜的花語」
3. **觀看動畫**：
   - 🎴 7 張卡片洗牌展開
   - ✨ 抽出一張卡片發光
   - 🔄 卡片 3D 翻轉
   - 🌸 花朵綻放（SSR 有金色特效！）
4. **查看結果**：
   - 3D 花朵主體展示
   - 花語、故事、今夜訊息
   - 推薦探索的鹽埕地點
   - SSR 獲得恭喜動畫
5. **自動收藏** - 卡片保存至圖鑑
6. **再抽一次** 或 **分享花語**

### 收藏圖鑑
1. **點擊右上角圖鑑按鈕** 📚
2. **查看收集進度**：總收集率、SSR 收集數、一般卡收集數
3. **切換分類**：全部 / SSR / 一般
4. **點擊已收集卡片** - 查看詳細資訊
5. **持續抽卡** - 解鎖完整 20 張圖鑑

### 收集目標
- 🎯 收集全部 20 張花語卡
- ⭐ 收集全部 5 張 SSR 稀有卡（各 5% 機率）
- 📚 達成 100% 圖鑑完成度

## 🎨 自訂花語資料

編輯 `src/data/flowers.json` 來新增或修改花語內容：

```json
{
  "id": 1,
  "flower": "花名",
  "meaning": "花語含義",
  "story": "花的故事",
  "message": "給使用者的訊息",
  "locations": ["地點1", "地點2", "地點3"],
  "color": "#HEX顏色碼",
  "model": "3d模型名稱"
}
```
